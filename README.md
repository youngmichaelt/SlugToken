## Welcome!

### What's SlugToken?

SlugToken is an ERC20 cryptocurrency on the Polygon network that you can stake to win transaction fees! What makes SlugToken so special is the staking + tax tokenomics model which incentivizes hodling and disincentivizes selling. Stake your SlugToken for the chance to win big!
